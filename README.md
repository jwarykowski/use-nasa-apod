# use-nasa-apod
React hook for the NASA astronomy picture of the day

## Getting started
```js
const apiUrl = 'https://test.com' // defaults to 'https://api.nasa.gov/planetary/apod'
const apiKey = 'secretKey'

const data = useNasaAPOD({ apiKey, apiUrl })
```
